---------------------------------------------SETTING UP THE BOT------------------------------------
1. Make sure NODE is installed @ nodejs.org make sure you add chocolatey during setup!
2. INSIDE the / directory of the bot run ~npm i package.json
3. Right click edit on `/cmds/addrole.js` File
& replace `let target = {some numbers}` with your role ID
3a. I HAVE MADE IT VERY NOTICIBLE WHERE THIS IS
4. In the / Directory of the bot run ~node bot.js
4a. THIS WILL give some arbitrary error about ./auth, ignore this! the bot SHOULD start normally
4b. Keep the prompt active to continue the bot running, check the prompt for user logs
4b1. ( NOTE The BOT is NOT made to handle a bunch of users running the 'dead chat' trigger at once, be advised)


If you want to make your own client:

Please also replace the the token in bot.js (edit /bot.js for this))
with your own from the discord dev portal @ discord.dev
